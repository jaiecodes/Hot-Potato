# Hot Potato Dating Sim 
#### Jaden Ambrocio Alcantara - CS 126 Final Project
![Title Screen](https://github.com/uiuc-sp21-cs126/final-project-Jeideun/blob/week-3/Title%20Screen.png)
## Purpose 
Hot Potato was a CS126 Final Project with the intent to entertain users. Hot Potato is a basic dating simulation where the main character can choose three different places on a US Potateo Farm to meet new patateos. The goal is to reach a certain love rating with the users desired potato.
